# Instagram Clone

## Approach
- **TailwindCSS for Styling**
- **Component-Based Structure**: The page is divided into three key components:
  - `MobileMockup`
  - `LoginForm`
  - `Footer`

## Setup and Running Locally
1. **Install dependencies**
   ```sh
   npm install
   ```
2. **Start the development server**
   ```sh
   npm run dev
   ```
